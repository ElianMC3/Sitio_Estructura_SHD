Elementos del Sitio

Los elementos del sitio son los componentes visuales y funcionales que conforman la interfaz de una aplicación web. Su correcta organización mejora la usabilidad, accesibilidad y experiencia del usuario.

Elementos de Navegación
  • Menú principal
  • Submenús
  • Breadcrumbs
  • Enlaces internos
  • Paginación
 Elementos de Contenido
  • Títulos y subtítulos
  • Párrafos informativos
  • Imágenes y multimedia
  • Tarjetas (cards)
  • Listas y tablas
 Elementos de Interacción
  • Botones
  • Formularios
  • Campos de búsqueda
  • Filtros
  • Modales y diálogos
 Elementos en Angular

En Angular, estos elementos suelen implementarse como componentes reutilizables, permitiendo mantener una estructura consistente en toda la aplicación.

Angular Material proporciona componentes listos para usar que siguen buenas prácticas de diseño y accesibilidad.

Buenas prácticas
  • Mantener coherencia visual
  • Evitar la sobrecarga de elementos
  • Usar componentes reutilizables
  • Priorizar la accesibilidad
