Breadcrumbs

Los breadcrumbs, también conocidos como “migas de pan”, son un elemento de navegación que muestra la ubicación actual del usuario dentro de la jerarquía de una aplicación o sitio web.

¿Para qué sirven?
  • Ayudan a entender la estructura del sitio
  • Facilitan la navegación entre niveles
  • Reducen la tasa de abandono
  • Mejoran la experiencia de usuario (UX)
 Tipos de Breadcrumbs
  • Jerárquicos: Muestran la estructura del sitio
  • Basados en rutas: Reflejan la navegación del usuario
  • Basados en atributos: Usados comúnmente en e-commerce
 Breadcrumbs en Angular

En Angular, los breadcrumbs se suelen generar a partir del sistema de rutas (Router), utilizando la configuración de rutas y metadatos como el título de cada sección.

Esto permite que los breadcrumbs se actualicen automáticamente cuando el usuario navega entre vistas.

Buenas prácticas
  • Mostrar siempre la ruta completa
  • No reemplazar la navegación principal
  • Usar separadores claros
  • Hacer cada nivel clickeable
